# Hotel Pickup / Check-in Card

This is the first printed card selected for ChinaReady.

## Why This Card First

- It supports the hotel delivery handoff before the traveler opens the kit.
- It helps hotel front desk staff understand why the parcel arrived before check-in.
- It gives the traveler a bilingual phrase to collect the parcel.
- It has low compliance risk compared with SIM cards, cash, medicine, or payment instructions.
- Its layout can be reused for hotel return cards, taxi cards, delivery cards, and emergency cards.

## Production Draft

| Item | Spec |
| --- | --- |
| Size | 90 x 54 mm wallet card |
| Print | Double-sided |
| Paper | 300-350 gsm matte card stock |
| Finish | Rounded corners optional |
| Front side | Hotel front desk holding message |
| Back side | Traveler pickup phrase |
| Variable fields | Guest name, hotel name, check-in date, order ID, support QR, map QR |
| Stage 1 method | Print blanks in bulk, then fill variables by handwriting or sticker labels |
| Packaging | One card inside the kit; one card or label outside the parcel |

## First Sample Batch

1. Print 20-50 blank cards.
2. Fill guest and hotel fields manually for test orders.
3. Put one copy in the box for the traveler.
4. Attach one copy or simplified sticker version outside the parcel for the hotel front desk.
5. Ask early customers whether the card helped them retrieve the parcel.
6. Ask hotels whether the wording was clear enough for front desk handling.
