Make it physical

Production notes for the first sample batch.

Print format

90 x 54 mm card, double-sided, 300-350 gsm matte paper, rounded corners optional.

Variable fields

Guest name, hotel name, check-in date, order ID, and optional support QR should be filled per order.

Batch method

For the first 20-50 boxes, print blanks in bulk and write variable fields by hand or sticker label.

Where it sits

One copy inside the kit for the traveler, one copy or sticker outside the parcel for hotel reception.

Supplier brief

Ask local print shops for business-card stock first. Upgrade to custom die-cut cards after demand is proven.

Next card

After this, reuse the layout for the hotel return address card and the taxi / ride-hailing phrase card.
